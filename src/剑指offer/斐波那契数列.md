#### 剑指offer 斐波那契数列

----

> 写一个函数，求斐波那契数列数列的第n项



#### 解法：动态规划

~~~go
func fib(n int) int{
	a,b,sum := 0,1,0
	for i := 0; i < n;i++{
		sum = (a+b) % 1000000007
		a =b
		b = sum
	}
	return a
}
~~~



#### 青蛙跳台阶问题

----

> 一只青蛙一次可以跳上1级台阶，也可以跳上2级台阶。求该青蛙跳上一个 n 级的台阶总共有多少种跳法。
>
> 答案需要取模 1e9+7（1000000007），如计算初始结果为：1000000008，请返回 1。

~~~go
func numWays(n int) int{
	a,b,sum := 1,1,0
	for i := 0; i < n;i++{
		sum = (a+b) % 1000000007
		a =b
		b = sum
	}
	return a
}
~~~
