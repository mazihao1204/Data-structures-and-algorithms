leetcode 11 盛水最多的容器

给你 n 个非负整数 a1，a2，...，an，每个数代表坐标中的一个点 (i, ai) 。在坐标内画 n 条垂直线，垂直线 i 的两个端点分别为 (i, ai) 和 (i, 0)。找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水

**解法：双指针**

~~~go
func maxArea(height []int) int{
	i := 0
	j := len(height) - 1
	res := 0
	for i < j{
		if height[i] < height[j]{
			area := (j-i) * height[i]
			i++
			if area > res{
				res = area
			}
		}else {
			area := (j-i)*height[j]
			j--
			if area > res{
				res = area
			}
		}
	}
	return res
}
~~~
